# What does $ \sqrt{i} ? $ - $ i = \sqrt{-1} $ - What does $ \sqrt{i} $ ? Use formula: - $ z = r \times e^{i\theta} = r(cos(\theta) + i \times sin(\theta))$ - $ z^n = r^n (e^{i\theta})^n = r^n (cos(n\theta) + i \times sin(n\theta))$ Calculate: - $ z = i $ - $ i = 0 + 1i \to r=1, \theta = \pi/2 $ - $ \sqrt{i} = (cos(\pi/2) + i \times sin(\pi/2))^{1/2} $ - $ = (cos(1/2 \times \pi/2) + i \times sin(1/2 \times \pi/2)) $ - $ = (cos(\pi/4) + i \times sin(\pi/4)) $ ## $ \sqrt{i} = \frac{1}{\sqrt{2}} + i \times \frac{1}{\sqrt{2}} $ Check: - $ ( \frac{1}{\sqrt{2}} + i \times \frac{1}{\sqrt{2}} )^ 2 = ( \frac{1}{\sqrt{2}} )^2 + 2 \times ( \frac{1}{\sqrt{2}} ) \times ( i \times \frac{1}{\sqrt{2}} ) + ( i \times \frac{1}{\sqrt{2}} )^2 $ - $ = \frac{1}{2} + i \times \frac{2}{2} - \frac{1}{2} = i $ # What does $ i^i ? $ - $ x = i^i $ - $ ln(x) = ln(i^i) = i \times ln(i) $ - $ e^{ln(x)} = x = e^{i \times ln(i)} $ Calculate $ ln(i) $: - $ e^{i \pi/2 } = (cos(\pi/2) + i \times sin(\pi/2)) = i $ - $ e^{i \pi/2 } = i \to ln(e^{i \pi/2 }) = ln(i) $ - $ i \pi/2 = ln(i) $ Continue: - $ e^{i \times ln(i)} = e^{i \times i \pi/2 } = e^{-\pi/2} $ ## $ i^i = e^{-\pi/2} = 0.20787957635 ... $ Is a real number!
