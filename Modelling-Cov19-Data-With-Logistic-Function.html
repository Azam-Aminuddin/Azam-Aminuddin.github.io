The Model¶

$ \Large f(t) = \frac{L}{1 + e^{-k(t-t_0)}} $
where:

  • L : The maximum capacity
  • k : The growth rate
  • $t_0$ : The inflection point of time
In [2]:
def f(L, k, t0, t):
    return L / ( 1 + np.exp( -k * (t-t0)))

The Fitting¶

We would like to guess the 3 parameters: L, k, $t_0$

In [6]:
def fObjective( x ):
    rmse = np.sqrt( np.square( theData.apply( lambda r : f( x[0], x[1], x[2], r.index ) - r )).sum().Confirmed / len(x) ) 
    return rmse, 0, 0
In [7]:
X = [8000, 0.5, 20] # an initial smart guess
X = scipy.optimize.fsolve( fObjective, X )

The Result¶

In [8]:
def DisplayResult( country, extrapolation = None ):
    display( Markdown("### %s" % country))
    display( Markdown("L=%.0f, k=%f, t0=%f, RMSE=%.0f" % ( X[0], X[1], X[2], fObjective( X )[0] ) ))
    plt.figure(figsize=(12, 6))
    plt.plot( theData )
    if extrapolation is None:
        plt.plot( theData.index, f(X[0], X[1], X[2], theData.index ) )
    else:
        plt.plot( extrapolation, f(X[0], X[1], X[2], extrapolation ) )
        
    plt.axvline( x = X[2], color="r", ls="--" )
    plt.xlabel('Days since first day')
    plt.ylabel('Confirmed')
    plt.legend(['Real', "Model"])
    plt.grid( True )
